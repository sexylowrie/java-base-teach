1.Collection与Collections有什么区别

    了解集合，首先要区分Collection和Collections的区别，虽然两者都位于java.util包下，但是两者还是有根本的区别。
    Collection是集合框架的根(ROOT)接口，Set，List，Queue，Deque都继承自此接口。
    而Collections是常用的集合工具类，提供了sort等方法。
